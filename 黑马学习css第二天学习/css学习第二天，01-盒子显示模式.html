块元素
常见块元素：div，p,ul.li,ol.li,dl,dt+dd/h
块元素特点，01-独占一行，02-设置宽高是可以生效的，如果不设置宽度，默认和父级盒子一样宽
块元素特点，03-给块级元素设置内外边距都生效
块元素特点，块元素里边可以嵌套任何元素，但是段落标签p里边不能嵌套div，因为浏览器解析的时候会将p嵌套div解析成两个p标签，一个div标签显示
行内元素
常见的行内元素：a，span，u，s，del，strong，em，i，ins
行内元素的特点，01-一行多个显示。02-行内元素设置宽高不会生效，默认内容撑开的。
行内元素的特点：03-设置内外边距上下不生效，左右生效
行内元素特点4：行内元素只能嵌套文本，图片，或者其他行内元素，但是超链接a里面可以嵌套块元素，我们最好将a转化为块元素，超链接a里面不能嵌套超链接a
行内块元素
常见行内块元素：img，input，button
行内块元素的特点，01-一行可以存多个，02-给行内块元素设置宽高和内外边距都生效，03-如果不设置宽度，默认是由内容撑开的
