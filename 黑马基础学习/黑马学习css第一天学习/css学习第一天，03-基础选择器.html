div{内容$}*6
{}表示生成的标签里面的内容
$表示生成从1开始的序列号
标签选择器 用标签的名称作为选择器，来选择元素 会将当前页面所有的该标签，全部选择，统一设置样式，实现不了其中某一个的差异化样式 实际开发中，不推荐直接使用，后期配合后代选择器使用
类选择器
类选择器 给标签用class=‘类名称’起名字 .类名{设置样式} 高级技巧：多类名调用 一个标签只能设置一个class属性，如果我们想要调用多个类名，我们可以在一个class里面直接以空格隔开，书写另一个类名称即可
id选择器
一个id选择器，一个页面只能出现一次。 第一步：在html结构中给对应的标签设置‘id属性=id名称’，调用css设置的样式 第二步：在css文件中使用#号，后边紧跟id名称，然后给对应的元素设置样式
通配符选择器
*号，统一选择选择html整个页面的全部元素
