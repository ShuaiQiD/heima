css三大特性，层叠性
一个标签的同一属性被多次使用，最后设置的一次会生效，其他的都会被层叠覆盖
遵从就近原则
css三大特性，继承性
有一些css子元素可以继承父级元素，不需要单独设置，控制文字的相关属性
超链接a元素不会继承父级盒子的color颜色，因为浏览器默认给a设置默认颜色样式，我们需要单独设置a的color颜色
标题元素除了h4以外所有h标签不会直接继承父级盒子的大小，font-size，因为他们是em的单位
css三大特性，权重优先级
不同的选择器选择元素权重大小不同，我们以基础选择器为参照，计算出我们自己选择器权重大小
基本计算
重点继承性和通配符选择器是没有权重的，如果当前元素有自己的样式，就不会继承父级样式，所以继承性和通配符选择器权重为0
权重叠加，权重叠加不会有进位的，每一个基础选择器只能在自己的位置叠加，如果超过10也不可以往前边进位。
权重优先级
继承性和通配符选择器的权重为0
标签选择器，伪元素的权重为0，0，0，1
类选择器，伪类选择器，属性选择器的权重为0，0，1，0
id选择器的权重为0，1，0，0
行内选择器的权重为1，0，0，0
设置样式！important提高权重，权重是无限大的，都提权重了吗，还是比较选择器
