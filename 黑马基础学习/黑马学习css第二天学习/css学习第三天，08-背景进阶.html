背景固定
背景固定语法：background-attachment
默认取值：scroll滚动。不滚动 fixed 固定
body的默认高度是内容撑开的，如果想要设置的和屏幕一样高，我们需要自己手动设置html和同body的宽度都是100% 这样就可以实现，body的高度和屏幕一样高了
背景图片尺寸：背景图片和屏幕一样大，需要背景图片铺满整个盒子的时候
语法background-size常用的取值
cover，等比例缩放直到铺满盒子
contain，等比例缩放，如果盒子图片不对称，图片在满足宽或者高的时候就停止缩放
背景属性综合写法
background属性没有书写顺序，没有的属性可以省略不写。属性和属性值之间用空格分开
background-color:颜色。background-image:url(url)：图片地址
background-repeat:背景是否平铺。background-position:水平 垂直
background：背景颜色 图片地址 是否平铺 水平位置 垂直位置
容易错误的点，如果综合写法和分开的某一个同时出现，综合写法一定放前面，会覆盖
背景的单独属性，不能设置符合取值，符合取值必须是background才可以
background-size写入综合属性的时候，必须要有background-position加上 / cover

背景颜色透明度语法：background-color:rgna(0,0,0,0.5)
