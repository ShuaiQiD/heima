background-color背景颜色
background-image插入背景图片,默认平铺
background-repeat背景图片平铺设置
repeat平铺，no-repeat不平铺
background-repeat:repeat-x水平方向平铺
background-repeat:repeat-y垂直方向平铺
background-position背景位置设置
background-position:水平 垂直
取值一个值的时候，垂直居中
可以去取值为方位名词，left，right，top，bottom，center
取值百分比的时候，为整个盒子为参照
上top，下button，左left，右right1
