文字大小控制font-size
文字大小控制font-size: 字体大小px。浏览器默认文字大小为16px
浏览器能识别最小的文字大小为12px
文字字体控制：font-family
文字字体控制：font-family：‘文字字体’。浏览器默认文字是微软雅黑
文字字体可以取多个值，多个值需要用英文逗号隔开。
多个值如果是汉字或者多个英文单词构成的英文，用引号隔开
文本粗细设置font-weight
文字加粗效果：font-weight：bold或者700。
文字不加粗效果：font-weight：normal或者400
细体文字设置100-300整百取值
文本不倾斜设置：font-style：normal
文本倾斜设置：font-style：italic
